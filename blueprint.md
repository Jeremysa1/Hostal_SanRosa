
# Proyecto de Formulario de Reserva

## Visión General

Este proyecto es una aplicación web de React que proporciona un formulario de reserva para un hotel o alojamiento. El formulario permite a los usuarios introducir su nombre, el número de personas, las fechas de llegada y salida, y su número de teléfono.

## Plan Actual

### Adaptar el Formulario de la Imagen a la Web

1.  **Crear el componente del formulario:** Reemplazar el contenido actual de `src/App.tsx` con un componente de React que contenga el formulario de reserva.
2.  **Añadir campos de entrada:** El formulario incluirá campos para:
    *   Nombre del huésped (campo de texto)
    *   Número de personas (campo numérico)
    *   Llegada (selector de fecha)
    *   Salida (selector de fecha)
    *   Celular (campo de texto)
3.  **Gestionar el estado del formulario:** Utilizar el hook `useState` de React para gestionar los valores de los campos del formulario.
4.  **Estilizar el formulario:** Añadir estilos CSS en `src/App.css` para que el formulario sea visualmente atractivo y se asemeje al diseño de la imagen proporcionada.
5.  **Añadir un botón de envío:** Incluir un botón para enviar el formulario.
